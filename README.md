# Tilt Ball Maze Game 


---

## 📱 Features

- Tilt-based movement using phone sensors  
   Player-controlled ball  
-  Maze with walls   
-  Collision detection with walls  
-  Built using Jetpack Compose Canvas  

---

## How It Works

- The app uses the **accelerometer sensor** to detect device tilt.
- Tilting the phone moves the ball:
  - Left/Right tilt = horizontal movement  
  - Forward/Backward tilt = vertical movement  
- The ball is constrained by:
  - Screen boundaries (walls)
  - Maze obstacles (rectangles)

---

## Technologies Used

- **Kotlin**
- **Jetpack Compose**
- **Canvas API**
- **Android SensorManager (Accelerometer)**
